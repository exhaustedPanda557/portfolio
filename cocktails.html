Bartending Help

Context

I created mobile Flutter app using asynchronous calls to an outside data source of cocktail information to display bartending help.

College students do not know how to make sophisticated alcoholic drinks and settle for gatorade spiked with alcohol. The mobile app is convenient and makes it easy to click one button and learn a margarita receipe.

Process

(click arrows on the slideshow for the progression)

First: I identified a JSON with cocktail information that the mobile app can make asynchronous calls to.

Second: I created a button Widget that when users click it asynchronously gets Strawberry or Smash Watermelon Margarita drink instructions (respectively).

Takeaway

It is most efficient to the user and creator if a buttons corresponds to just one piece of data from the JSON returned in an API call (i.e. each drink has its own button to get the bartending instructions).
